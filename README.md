# Claude Timer (AI Usage Tracker)

Claudeのメッセージ制限が「あと何分でリセットされるか」を、ブラウザの右上にちょこんと表示してくれる拡張機能です。

## 🐣 なにができるの？

1.  **残り時間をバッジでお知らせ**: ブラウザのツールバーにあるアイコンの上に、リセットまでの残り時間が表示されます。
2.  **ワンクリックで確認**: アイコンをクリックすると、すぐにClaudeの「使用量」ページが開きます（すでに開いている場合はそのページに飛びます）。
3.  **完了をお知らせ**: リセット時間が来ると、バッジが緑色になって「OK」と教えてくれます。

## 🛠️ インストール方法（使い方）

この拡張機能はストアには公開していません。以下の手順であなたのブラウザに入れてみましょう！

### 1. ファイルを準備する
1.  このページの緑色の **[Code]** ボタンを押して、**[Download ZIP]** をクリックします。
2.  ダウンロードしたZIPファイルを解凍（展開）して、フォルダの状態にします。

### 2. ブラウザに読み込ませる
1.  Google Chrome（またはEdgeなど）を開きます。
2.  アドレスバーに `chrome://extensions/` と入力してエンターキーを押します（または、右上の「︙」メニュー > 拡張機能 > 拡張機能を管理）。
3.  画面右上の **「デベロッパーモード」** というスイッチを **ON** にします。
4.  現れた **[パッケージ化されていない拡張機能を読み込む]** ボタンをクリックします。
5.  さきほど解凍したフォルダ（`claude_timer` フォルダ）を選択します。

これでインストール完了です！🎉

## 📖 使い方

1.  ブラウザの右上に「AI Usage Tracker」のアイコン（ジグソーパズルのピースのようなアイコンの中に隠れているかもしれません）が表示されます。ピン留めしておくと便利です。
2.  アイコンをクリックしてください。Claudeの「設定 > 請求 > 使用量」ページが開きます。
3.  ページが開くと、自動的に「リセット時間」を読み取ります。
4.  アイコンの上に「2:30」（あと2時間30分）のように数字が表示されれば成功です！

---

### 注意事項
- Claudeの画面デザインや言葉が変わると、時間が読み取れなくなることがあります。
- このツールは非公式の学習用プロジェクトです。
